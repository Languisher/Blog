## 虚拟化

计算机操作系统 给每一个进程一个假象：它是系统中唯一正在执行的程序，独占计算机。

## 程序与进程

计算机操作系统是一个状态机，
- 在每一个 step 都会执行其下面的进程的一个指令

而对于每个进程来说也是一个状态机，
- 在每一个 step 都会执行其一个指令


```python
x = ... # receive x from input
for (i ...):
	if (x > 0):
		...
	else:
		...
```


**程序**是进程的静态描述
- 描述了所有可能的程序状态。这意味着对于不同的输入，都有其对应的应用处理方式
- 程序（动态）运行起来，就成了进程——会有其内存空间和寄存器，以及进行系统调用等

**进程**是程序的运行时状态随时间的演进
- 进程状态有寄存器和内存
- 但除此之外，进程还有一些静态属性，例如进程编号，由操作系统管理
